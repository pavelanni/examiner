LLM-based conversational assessment system

Open-source · Go · Any LLM backend

1 / 7
The problem

Traditional exams fall short

  • — Written tests only capture surface-level recall, not true understanding
  • — Oral exams are powerful but don't scale — one professor, many students
  • — No opportunity for follow-up questions that probe deeper comprehension
  • — Grading open-ended answers is time-consuming and inconsistent
2 / 7
The idea

What if every student could have a one-on-one oral exam — with an AI that asks follow-up questions?

Examiner combines the depth of oral exams with the scalability of automated testing. The LLM probes understanding through conversation, then grades against a rubric.

3 / 7
Key features

What it does

💬

Conversational flow

LLM asks follow-up questions to probe deeper understanding (configurable limit per question)

⚖

Automated grading

Scores each answer against a rubric and model answer, produces an overall grade

🧑‍🏫

Teacher review

Faculty can adjust per-question scores, add comments, and finalize the grade

🌐

Multilingual

UI in English and Russian; adding a new language requires only a JSON file
